# CastNotetrackEditor
Notetrack Viewer and Editor for Blender

## Installation
To update, only `cast_notetrack_editor.py` and `__init__.py` need to be replaced.

For a clean install:
1. Download this repo as a zip
2. Open Blender
3. Go to **Edit → Preferences → Add-ons → Install**
4. Select the downloaded zip file

> **Note:** If updating, it is recommended to uninstall the previous version first.

> **Disclaimer:** Tested on Blender 3.6 only.
